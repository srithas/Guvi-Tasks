READ ME

TO RUN CODE:

Open folder in IDE
Start XAMPP: Apache and MySQL
In IDE run: EmployeeManagementApplication.java
On browser go to http://localhost:8060/

SET UP DATABASE:

Start XAMPP: Apache and MySQL
On browser go to http://localhost/phpmyadmin/index.php
Create new database name testemp
Create new table employee with fields: id(PRIMARY KEY, int(11), AUTO_INCREMENT), firstname(varchar(100)), lastname(varchar(255)), email(varchar(255)), position(varchar(255)), salary(decimal(10,2))

OVERVIEW:

Click Add New Employee button
Enter employee details and save
Click view to view employee details
Click edit to edit employee details
Click delete to delete employee
